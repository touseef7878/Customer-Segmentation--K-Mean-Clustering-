# 🛍️ Customer Segmentation (Mall Customers)

## 📌 Project Overview

This project performs **Customer Segmentation** using:

* **KMeans Clustering**
* **DBSCAN (Bonus)**

We use customer demographics (**Age, Income, Spending Score**) to group shoppers into meaningful clusters.
